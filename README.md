1.Power BI Desktop
2.Power BI Service
3.Power BI Mobile  
( 4th one would be PowerBI Gateway)

4.What is the difference between Power BI Desktop and Power BI Service?
Answer:PowerBI desktop is windows based applicaton for creating report, visuals.
Power BI Service is cloud based platform for publishing and sharing reports.
